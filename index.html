FrequentlyAsked Questions

1. The difference between var, let, and const
The difference between var, let, and const are: we can redeclare a variable with var but we can not redeclare a variable with let and const. var is hoisted with undefined but let and const are hoisted with TDZ. There are also differences between let and const. We can update a variable with let but we can not update a variable with const.
2. The difference between map(), forEach(), and filter()
1. map() returns a new array after iterate over an array. 2. foreach() iterate over an array but does not returns a new array. 3. filter() returns a new array after filtering over an array.
3. Explain arrow functions and how they are different from regular functions
arrow functions are the modern way to write functions in JavaScript. But arrow functions does not have access to this keyword like regular functions have.
4. How JavaScript Promises work
JavaScript Promises are used to handle asynchronous operations. The promise can be resolved and rejected. We can use .then() to access the promise if the promise is resolved. And we can use .catch() to catch exceptions that are thrown by the promise.
5. How closures work in JavaScript
If we return a function inside another function, the inner function will have access to the outer function's variables. Even after the outer function has finished executing the inner function will remember the outer function's variables.
